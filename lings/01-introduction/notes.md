# Introduction
Welcome to my Rust learning repository. Here, I learn Rust and write notes with example code. Most of the information here is based on [The Rust Programming Language](https://doc.rust-lang.org/stable/book/) book. Let's begin.

## What is Rust
Rust is a modern, general-purpose, ahead-of-time compiled programming language that focuses on **safety**, **performance**, and **concurrency**.

## Rust advantages
Rust helps developers write **fast** and **reliable** software. It provides control over **low-level** details, such as **memory management**, without the traditional *hassle* that comes with such control.
