# Introduction
Welcome to my Rust learning repository. Here I learn Rust and write notes with example code. Most of the information here is based on [The Rust Programming Language](https://doc.rust-lang.org/stable/book/) book. Okay, let's begin!

## What's Rust
Rust is a modern *general-purpose* programming language focusing on **safety**, **performance**, and **concurrency**.

## Rust benefits
The Rust programming language helps write *faster*, *more reliable* software. Rust gives the option to control **low-level** details, such as memory usage, without all the traditional hassle coming with such control.
